## Overview
    A documentation on learning Microsoft 365 for SOC. Exploring core concepts of Entra ID and M365, the threats targeting them, and how to monitor their logs and activity as a SOC analayst.

# Learning Objectives
    Build a foundational understanding of how these platforms work, then explore the biggest threats and attack techniques that target them.
    The module covers monitoring, walking through the key logs, events, and signals from Entra ID, Exchange, SharePoint, and Intune that matter most from a SOC perspective.
    Hands-on work with a Splunk instance loaded with real platform logs, putting theory into practice as we investigate suspicious activity across these lab environments.
